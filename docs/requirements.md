> 版本: v2.0 | 日期: 2026-06-24 | 维护: Dayong Zhang & CodeBanana

## 1. 产品概述

### 1.1 产品定义
**HH小程序** — AI 驱动的身体健康行动引导应用。

### 1.2 核心问题
信息过载时代，人们**到处看健康内容但不知道怎么行动**，以及**知道该做但坚持不下去**。

### 1.3 解决方案
- **行动引导**：不是给你更多内容看，而是告诉你**现在该做什么**，跟着做就行
- **AI 个性化**：每个人进来看到不同的推荐，基于你的身体状况和问题
- **正反馈系统**：即时反馈让你每次做完都有成就感，长期反馈让你看到真实改变

### 1.4 目标用户
- 久坐不动、手机依赖的上班族/学生
- 有亚健康问题（肩颈痛、腰痛、体态差、脑雾）的人
- 情绪低落、压力大但不知道从哪里开始改善的人
- 到处收藏健康内容但从来不行动的人

## 2. 核心定位

### 2.1 主次关系
| 优先级 | 内容 | 占比 |
|--------|------|------|
| **核心** | 科学系统的居家运动跟练视频 | 70% |
| 辅助 | 健康小知识（身体科普） | 15% |
| 辅助 | 冥想跟练 | 15% |

### 2.2 不是什么
- ❌ 不是冥想 App（冥想只是辅助）
- ❌ 不是健身 App（不追求强度和身材）
- ❌ 不是知识付费（不是"看文章"，是"跟着做"）
- ❌ 不是打卡工具（打卡只是手段，行动才是目的）

### 2.3 是什么
- ✅ **行动引导**：打开就知道该做什么，跟着做就行
- ✅ **身体健康**：解决身体层面的实际问题（久坐、体态、脑雾）
- ✅ **简单有效**：每个动作科学但有依据，不花哨不复杂
- ✅ **千人千面**：AI 根据你的情况推荐最适合你的行动

## 3. 用户痛点与场景

### 3.1 核心痛点

| 痛点 | 表现 | HH 怎么解决 |
|------|------|------------|
| 信息过载 | 收藏了几百个健康视频但没练过一个 | AI 只推荐**1个最适合的行动**，不让你选 |
| 不知道怎么动 | 想改善肩颈但不知道做哪个有用 | 基于你问题的精准推荐，不是泛泛的"运动列表" |
| 坚持不下去 | 做了2天就放弃了 | 即时正反馈 + 长期正反馈机制 |
| 久坐不动 | 坐了一整天没动过 | 定时提醒 + 微行动（2分钟也能做） |
| 体态差 | 圆肩驼背脖子前倾 | 系统化的体态改善路径 |
| 脑雾/情绪差 | 注意力下降、情绪低落 | 运动+呼吸+冥想的组合方案 |

### 3.2 典型场景

**场景 A — 久坐上班族**
> 上午10点，坐了2小时没动过。HH 推送提醒：「起来动一下？」打开后推荐 3 分钟肩颈拉伸跟练，做完立刻看到「你的肩颈今天第1次活动了」✨

**场景 B — 体态焦虑者**
> 刷手机看到自己照片，圆肩驼背。打开 HH，首页推荐「7天体态改善计划」，今天是第3天：靠墙天使 2组×10次，跟着视频做就行。

**场景 C — 情绪低落不知道干嘛**
> 心情不好但不知道做什么。HH 首页：「今天做个 5 分钟呼吸+轻运动？」跟着做，做完弹出心情记录，比开始时好了一点。

**场景 D — 到处收藏不行动**
> 收藏了几十个运动视频但从来没练过。HH 打开就一个推荐：「现在做这个」，不需要你选，跟着做就行。

## 4. 内容体系

### 4.1 运动跟练内容（核心 70%）

**内容形式：短视频跟练（30秒-10分钟）**

| 类别 | 内容 | 时长 | 难度 |
|------|------|------|------|
| **微行动** | 站起来活动一下、转转脖子、伸个懒腰 | 30秒-2min | 零门槛 |
| **久坐急救** | 肩颈拉伸、腰背放松、手腕放松、眼睛放松 | 3-5min | 入门 |
| **体态改善** | 圆肩矫正、驼背改善、骨盆前倾、脖子前倾 | 5-10min | 入门-中级 |
| **活力唤醒** | 晨间唤醒操、办公室活力操、午后提神 | 3-5min | 入门 |
| **压力释放** | 释放紧张的运动、拍打放松、抖动放松 | 3-5min | 入门 |
| **核心稳定** | 核心激活、腰腹力量、稳定性训练 | 5-10min | 入门-中级 |
| **柔韧性** | 全身拉伸、髋部打开、肩胸打开 | 5-10min | 入门-中级 |
| **睡前放松** | 睡前拉伸、助眠瑜伽、呼吸放松 | 5-10min | 入门 |

**每个跟练视频的结构：**
```
1. 问题说明（5秒）：这个练习解决什么问题
2. 动作演示：真人示范，有节拍/倒计时
3. 跟练模式：屏幕倒计时 + 语音引导
4. 完成反馈：即时正反馈（见第5节）
```

### 4.2 健康小知识（15%）

| 类型 | 示例 |
|------|------|
| 身体科普 | 「为什么久坐腰痛？你的腰椎在承受什么」 |
| 行动贴士 | 「3个动作让你告别圆肩，第2个最简单」 |
| 常见误区 | 「拉伸不是越痛越好，正确拉伸是什么感觉」 |

**知识内容特点：** 短、实用、看完就知道做什么。不是长文章，是 30 秒能看完的卡片。

### 4.3 冥想跟练（15%）

| 类型 | 内容 |
|------|------|
| 呼吸练习 | 4-7-8 呼吸、箱式呼吸、腹式呼吸 |
| 身体扫描 | 配合运动后的身体觉察 |
| 正念冥想 | 简短引导（3-10分钟） |
| 动态冥想 | 行走冥想、伸展冥想（和运动结合） |

## 5. 正反馈系统

### 5.1 即时正反馈（每次做完立刻感受到）

| 触发 | 反馈 | 例子 |
|------|------|------|
| 完成跟练 | 身体状态改善提示 | 「你的肩颈今天第1次活动了」✨ |
| 完成跟练 | 能力值增长 | 「核心稳定性 +3 📈」 |
| 完成跟练 | 解锁成就 | 「🏆 久坐杀手：连续3天做久坐急救」 |
| 首次尝试 | 鼓励 | 「太棒了！迈出第一步 🌟」 |
| 微行动 | 即时激励 | 「虽然只做了1分钟，但比不动强100倍 💪」 |
| 打破久坐 | 久坐中断 | 「你打破了2小时的久坐！身体在感谢你 🙏」 |
| 连续打卡 | streak 提醒 | 「🔥 连续第7天！你的身体在适应改变」 |

**即时反馈的视觉设计：**
- 完成动画（星星/光芒扩散）
- 能力值上涨的数字动画
- 进度条增长
- 成就徽章解锁弹出

### 5.2 长期正反馈（持续使用后看到的变化）

| 周期 | 反馈 | 例子 |
|------|------|------|
| 每周 | 周报 | 「本周你活动了 47 分钟，比上周多 15% 📊」 |
| 每月 | 身体变化 | 「坚持30天，你的体态改善分数从 42→58」 |
| 里程碑 | 阶段性成就 | 「🏆 100分钟运动达成！你的身体比你以为的更强」 |
| 习惯养成 | 行为转变 | 「你从'偶尔做做'变成了'每周5次' 🌱」 |
| 能力提升 | 新内容解锁 | 「你的核心稳定性达到 Lv.3，解锁进阶训练！」 |
| 周期评估 | 重新评测 | 「30天体态复测：圆肩改善 30%，继续保持！」 |

### 5.3 能力值系统（游戏化）

```
身体能力面板
├── 🦴 体态健康    Lv.2 ██████░░░░ 58/100
├── 💪 核心稳定    Lv.1 ████░░░░░░ 35/100
├── 🧘 柔韧性      Lv.1 █████░░░░░ 42/100
├── ⚡ 活力值      Lv.3 ████████░░ 72/100
└── 🧠 身心平衡    Lv.1 ███░░░░░░░ 28/100
```

- 每次完成对应类型的跟练，对应能力值 +1~5
- 能力值达到阈值升级，解锁新内容
- 能力值长期不练会缓慢衰减（但衰减速度很慢，不会让人焦虑）

## 6. 千人千面设计

### 6.1 首页核心原则：**不让你选，告诉你做什么**

首页只展示 **1-3 个推荐行动**，不是一堆内容让你挑。

```
首页结构：

┌────────────────────────────┐
│  ☀️ 早上好                  │  ← 个性化问候
│  今天你的肩颈需要关心        │  ← 基于画像的今日焦点
├────────────────────────────┤
│  ┌──────────────────────┐  │
│  │  🎯 今日行动          │  │  ← 主推荐（1个大卡片）
│  │  5分钟肩颈拉伸跟练     │  │
│  │  「你的肩颈3天没活动了」│  │
│  │  [ 开始跟练 → ]       │  │
│  └──────────────────────┘  │
│                             │
│  💡 或者试试                │  ← 备选（1-2个小选项）
│  · 2分钟站立活动             │
│  · 3分钟呼吸练习             │
├────────────────────────────┤
│  🔥 连续3天 · 活力值 72      │  ← 正反馈展示
│  ████████░░                 │
├────────────────────────────┤
│  📊 你的身体变化             │  ← 长期反馈
│  体态健康 +12 本周           │
└────────────────────────────┘
```

### 6.2 推荐逻辑（运动优先）

| 维度 | 权重 | 说明 |
|------|------|------|
| 身体问题匹配 | 0.35 | 你有什么问题→推什么运动 |
| 时段匹配 | 0.25 | 早上→唤醒；工作中→久坐急救；睡前→放松 |
| 行动频率 | 0.20 | 好久没动→推微行动；经常动→推进阶 |
| 新鲜度 | 0.10 | 没做过的优先 |
| 热度 | 0.10 | 大家觉得有效的优先 |

### 6.3 不同用户的首页差异

**用户 A — 久坐程序员（肩颈痛）**
```
🎯 今日行动
5分钟肩颈拉伸跟练
「你的肩颈已经4天没活动了」

或者试试：
· 2分钟站立活动（零门槛）
· 3分钟呼吸练习（减压）

🔥 连续2天 · 体态健康 42
```

**用户 B — 体态焦虑的新手妈妈**
```
🎯 今日行动
7天体态改善 · 第4天
靠墙天使 2组×10次

或者试试：
· 3分钟圆肩矫正（入门）
· 5分钟睡前拉伸

🔥 连续4天 · 体态健康 55
```

**用户 C — 知道要动但总拖延的上班族**
```
🎯 今日行动
1分钟站起来活动一下
「只要1分钟，比不动强100倍」

或者试试：
· 3分钟午休拉伸
· 2分钟眼保健操

💪 今天还没动过，动一下就好
```

## 7. 用户画像

### 7.1 冷启动问卷

**第1步：你有什么身体问题？（多选）**
```
🦴 肩颈疼痛/僵硬
🫄 腰背不适
🧍 体态差（圆肩/驼背/骨盆前倾）
🧠 脑雾/注意力差
😰 压力大/焦虑
😴 睡眠差
👀 眼睛疲劳
🏃 久坐不动
```

**第2步：你能接受的运动时长？（单选）**
```
⏱️ 1-3分钟（微行动，随时能做）
⏱️ 5分钟（短练习）
⏱️ 10分钟（完整练习）
```

**第3步：你的运动习惯？（单选）**
```
🛋️ 基本不动（从零开始）
🚶 偶尔动动（想养成习惯）
🏃 有运动习惯（想针对性改善）
```

### 7.2 画像更新

| 事件 | 更新内容 |
|------|---------|
| 完成跟练 | 能力值 +N，对应类别权重提升 |
| 跳过推荐 | 该类别权重微降 |
| 久坐提醒响应 | 久坐敏感度调整 |
| 每周评估 | 能力值变化趋势，推荐策略调整 |
| 30天复测 | 问题标签重新评估 |

## 8. 技术栈

| 层级 | 选型 | 说明 |
|------|------|------|
| 前端 | 原生微信小程序 | 直接用微信能力 |
| 后端 | Node.js + TypeScript | 与小程序生态统一 |
| AI | 火山引擎（豆包 LLM + 火山 TTS） | 国内服务，后期可换 |
| 数据库 | Supabase (PostgreSQL) | 结构化数据 + 向量查询 |
| 缓存 | Redis | 推荐结果、用户状态热数据 |
| 视频 | 微信视频组件 + OSS | 跟练视频托管 |

## 9. MVP 开发路线

### Phase 1 — 基础骨架 + 运动跟练 (2-3 周)
- [ ] 小程序项目初始化 + 页面骨架
- [ ] 后端项目搭建
- [ ] Supabase 数据库 + 核心表
- [ ] 微信登录 + 用户基础表
- [ ] **运动跟练视频列表 + 播放器**
- [ ] 冷启动问卷
- [ ] 首页"今日行动"推荐（规则驱动）

### Phase 2 — 正反馈 + 个性化 (2-3 周)
- [ ] 即时正反馈系统（完成动画 + 能力值 + 成就）
- [ ] 能力值面板
- [ ] 推荐引擎 v1（基于问题+时段+频率）
- [ ] 久坐提醒
- [ ] 健康小知识卡片
- [ ] 每周周报

### Phase 3 — AI 增强 + 长期反馈 (2-3 周)
- [ ] AI 个性化推荐（火山引擎 LLM）
- [ ] 30天体态复测
- [ ] 长期身体变化追踪
- [ ] 冥想跟练模块
- [ ] 能力值升级解锁内容

### Phase 4 — 打磨上线 (1-2 周)
- [ ] UI/UX 优化
- [ ] 视频加载优化
- [ ] 数据埋点
- [ ] 提审上线
