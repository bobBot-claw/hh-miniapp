## 8. 推荐打分公式

### 8.1 运动推荐评分

```
Score = ProblemMatch × 0.35    -- 问题匹配度 (最重要)
      + TimeMatch × 0.25       -- 时段匹配度
      + FrequencyMatch × 0.20  -- 行动频率匹配
      + Freshness × 0.10       -- 新鲜度
      + Popularity × 0.10      -- 完成率热度
```

**各维度计算：**

| 维度 | 计算 | 范围 |
|------|------|------|
| ProblemMatch | 用户问题标签 ∩ 运动问题标签 / 用户问题标签总数 | 0~1 |
| TimeMatch | 当前时段 ∈ 运动场景标签 ? 1.0 : 0.3 | 0.3~1 |
| FrequencyMatch | 好久没动→微行动1.0/进阶0.3；经常动→进阶1.0/微行动0.5 | 0.3~1 |
| Freshness | 近7天未做过 ? 1.0 : 0.3 | 0.3~1 |
| Popularity | 该运动完成率 / 最高完成率 | 0~1 |

### 8.2 久坐触发优先

```
IF 久坐时间 >= 120分钟:
  久坐急救类运动 Score × 1.5
  微行动类运动 Score × 1.3

IF 久坐时间 >= 60分钟:
  久坐急救类运动 Score × 1.2
  微行动类运动 Score × 1.1
```

### 8.3 新手保护

```
IF exercise_habit == 'sedentary':
  难度 > 入门 的运动 Score × 0.3
  时长 > 5min 的运动 Score × 0.5
  微行动 Score × 1.3

IF exercise_habit == 'occasional':
  难度 > 初级 的运动 Score × 0.5
```
