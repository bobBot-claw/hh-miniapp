# 私教形象设计规范

## 形象概述
3D 渲染风格的卡通女性私教角色，亲和力与专业感兼顾。用于所有跟练视频、引导页、反馈动画等场景。

## 外观设定

### 面部
- **风格：** Pixar 质感 3D 卡通，非真人
- **肤色：** 白皙/瓷白色
- **脸型：** 圆润友好
- **表情：** 温暖鼓励的微笑，大方露齿
- **眼睛：** 大而有神，略带高光
- **鼻子：** 小巧
- **脸颊：** 淡淡红晕

### 发型
- **短高马尾**，深色发，束在头顶高位
- 马尾长度只到后颈，干练利落
- 运动时不遮挡视线

### 体型
- 超模比例：小头、长腿、修长优雅
- 头身比约 1:9
- 运动型苗条身材，但偏修长而非健壮

## 服装
- **上衣：** 紧身拉链运动外套，**紫色→青色渐变**（从肩部紫 → 下摆青）
- **Logo：** 胸前小 "HH" 白色标志
- **裤子：** 黑色紧身运动裤，大腿处小 "HH" 标志
- **鞋子：** 白色极简运动鞋

## 色彩体系
| 元素 | 色值 |
|------|------|

| 头发 | 深棕 #2c1810 |

- 白色圆润机身，紫青渐变胸牌
- 黑色荧屏脸，青色发光笑脸
- 亲和力强，完全无审核问题
