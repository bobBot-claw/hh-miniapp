# 慢慢变好 — 微信小程序

> 每天一个极简行动，每次完成揭示一幅治愈插画。

## 产品定位

极简行动 + 治愈世界 wellness 体验。不做信息过载的健身 App，不做让你焦虑的习惯打卡。每天给你一个行动，做完就行。

## 5 个页面

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | pages/home | 今日行动 + ✦光点 + CTA |
| 行动 | pages/action | 倒计时进度环 + 步骤引导 |
| 揭示 | pages/done | 插画从模糊到清晰 + "今天有了光" |
| 世界 | pages/world | 12 个主题世界网格 |
| 情绪 | pages/mood | 5 种天气 + 1 个词 |

## 导航方式

流程式导航（无 TabBar）：
```
首页 → 行动 → 揭示 → 世界
 ↕                      ↕
情绪 ←——————————————————→ 首页
```

## 设计系统

- **暖白：** #FCFCFD
- **翠绿：** #147D45
- **暖金：** #C9A66B
- **深暗：** #2A2820（首页/行动页背景）
- **图标：** 单色线条（不用 emoji）
- **字体：** 系统默认，不混排

## 15 个行动

9 个时间轴（晨/工/夜 × 3）+ 6 个问题轴（晕/困/核心/体态/颈/眼）

## 12 个世界

| 编号 | 名称 | 图标类型 | 状态 |
|------|------|----------|------|
| 1 | 春之森林 | forest | ✅ 开放 |
| 2 | 夏之海边 | sea | ✅ 开放 |
| 3 | 秋之窗边 | window | ✅ 开放 |
| 4 | 冬之暖屋 | warm | ✅ 开放 |
| 5 | 茶与书 | tea | ✅ 开放 |
| 6 | 夜与星 | night | ✅ 开放 |
| 7-12 | 扩展世界 | default | 🔒 锁定 |

每个世界 30 张治愈插画，完成行动揭示 1 张。

## 彩蛋机制

- 未揭示：显示预渲染模糊版 (blur_XX.png)
- 揭示后：opacity 切换到清晰版 (clear_XX.png)
- 30/30 完成 → 大图奖励
- 12/12 世界 → 年度全景

## 技术栈

- **前端：** 原生微信小程序
- **状态：** localStorage (wx.setStorageSync)
- **数据：** utils/actions.js（纯本地，无后端依赖）
- **AppID：** wx91db22c766984692

## 15 条红线

1. 不用 emoji 做主图标
2. 不混排多字体
3. 不用纯黄/暖米做背景
4. 不用纯黑/重阴影
5. 不做仪表盘
6. 不做社交排行
7. 不做推送
8. 不用鸡血语言
9. 不做连续打卡
10. 不做惩罚
11. 不做签到游戏化
12. 不做内容消费
13. 不做强制日活
14. 首页不做百分比/进度
15. 不做倒计时数字

## 开发

1. 用微信开发者工具打开 `client/` 目录
2. 项目自动加载，首页为 pages/home
3. 插画资源暂未就绪，使用 CSS 渐变 fallback
