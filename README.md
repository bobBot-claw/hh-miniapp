# 慢慢变好 — 微信小程序

> 每天一个极简行动，每次完成揭示一幅治愈插画。

## 产品定位

极简行动 + 治愈世界 wellness 体验。不做信息过载的健身 App，不做让你焦虑的习惯打卡。每天给你一个行动，做完就行。

| 页面 | 路径 | 说明 |
|------|------|------|

### 行动结构

每个行动包含：
- **5~6 步引导**：每步有动作说明 + 感受提示（hint）
- **feeling**：做完当下的体感（揭示页展示）
- **benefit**：长期收益说明（揭示页展示）
- **depth**：light / medium / deep 三级深度
- **tags**：标签系统，用于问题轴匹配
