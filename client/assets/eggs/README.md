## 目录结构
```

  {worldId}/

    clear_01.png ... clear_30.png — 清晰版（揭示后显示）
```

## 插画要求
- **风格：** 治愈系插画（非照片），光遇/纪念碑谷风格
- **尺寸：** 750×1334px（全屏展示）
- **模糊版：** 高斯模糊 radius=20 + 亮度降低15% + 饱和度降低20%
- **清晰版：** 原始插画
- **格式：** PNG，优化后每张 < 200KB
- **总计：** 12世界 × 30张 × 2版本 = 720张
