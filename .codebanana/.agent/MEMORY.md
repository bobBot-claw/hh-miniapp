# MEMORY.md — Project Knowledge

## Project: HH小程序
- **类型:** WeChat Mini Program（微信小程序）
- **定位:** 身体健康行动引导应用（运动为主，静养为辅）
- **核心问题:** 信息过载 → 不知道怎么行动 + 坚持不下去
- **核心内容:** 科学系统的居家运动跟练视频 + 静养冥想练习
- **核心理念:** AI 驱动个性化，打开就知道该做什么，不让你选
- **正反馈:** 即时反馈（完成动画+能力值+成就）+ 长期反馈（周报+月报+复测）

## 重要方向变更
- **v1.0 方向:** 冥想为主的健康生活习惯管理（类似潮汐）
- **v2.0 方向 (当前):** 运动为主的身体健康行动引导 — 解决久坐/体态/脑雾等身体层面问题
- **变更原因:** huanhuan 提出纠正方向，从冥想优先改为运动优先
- **最新调整 (huanhuan):** 砍掉「发现」模块，换成「静养」模块，专门放冥想练习内容
- **当前 TabBar:** 首页 / 运动 / 静养 / 我的

## 技术栈决策
- **前端:** 原生微信小程序
- **后端:** Node.js + TypeScript
- **AI API:** 火山引擎 (豆包 LLM + 火山 TTS)，后期可换
- **数据库:** Supabase (PostgreSQL)
- **缓存:** Redis
- **视频:** 微信视频组件 + OSS
- **音频:** BackgroundAudioManager (小程序后台播放)
- **Git:** https://github.com/bobBot-claw/hh-miniapp

## 架构分层
- Client (小程序端) → API Gateway (BFF) → 微服务 (用户/内容/AI引擎) → 数据层
- AI 引擎四大模块: 推荐引擎 / 反馈引擎 / 动态内容生成 / 适应性决策
- 推荐引擎: 问题匹配(0.35) + 时段(0.25) + 行动频率(0.20) + 新鲜度(0.10) + 热度(0.10)
- 用户画像: 问题标签 + 运动习惯 + 能力值面板(5维) + 行为向量

## 能力值系统
- 🦴 体态健康 / 💪 核心稳定 / 🧘 柔韧性 / ⚡ 活力值 / 🧠 身心平衡
- 每次完成对应运动增长，达到阈值升级解锁新内容

## MVP 优先级
- P0: 运动跟练视频+播放器, 冷启动问卷, 今日行动推荐, 即时正反馈
- P1: 能力值面板, 久坐提醒, 推荐引擎v1, 每周周报
- P2: AI 个性化推荐, 30天复测, 静养冥想模块, 个性化计划生成
